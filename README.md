## Установка

### Системные требования
*   Среда для эксплуатации OC Android

### Пошаговая инструкция
1.  Зайдите в Releases
2.  Скачайте APK файл
3.  Установите приложение на смартфон

## Использование

Подтвердите своё нынешнее местоположение чтобы увидеть прогноз погоды в вашем городе, если вам недостаточно то можете ввести нужный вам город и увидеть прогноз погоды введённого вами города
